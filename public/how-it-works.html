Comment ça marche ?

Nimwema permet d'envoyer, de demander et d'utiliser des bons d'achat alimentaires en RDC. Voici une vue d’ensemble, étape par étape, pour chaque type d’utilisateur.

1. Demander un bon d'achat

Si vous avez besoin d’un soutien, vous pouvez créer une demande de bon depuis la page « Demander un bon d'achat ».

  1. Remplissez vos informations
    Indiquez votre nom et votre numéro de téléphone au format +243.
  2. Choisissez le type de demande
    Demander à un numéro connu (à une personne précise) ou Liste d'attente (48h) pour apparaître dans la file d’attente des donateurs.
  3. Validez votre demande
    Votre demande reste visible pendant 48 heures. Passé ce délai, elle expire automatiquement.
  4. Suivez vos demandes
    Vous pouvez consulter, mettre à jour ou supprimer vos demandes encore actives.

2. Envoyer un bon d'achat

En tant que donateur (particulier, entreprise, église…), vous pouvez envoyer un bon d’achat à un ou plusieurs bénéficiaires.

  1. Choisissez le montant et la quantité
    Sélectionnez un montant prédéfini ou saisissez un montant personnalisé, puis indiquez le nombre de bons à envoyer.
  2. Sélectionnez les bénéficiaires
    Vous pouvez :
    • Entrer un numéro unique.
    • Coller plusieurs numéros (jusqu’à une limite définie).
    • Choisir dans la liste d’attente des demandes reçues.
  3. Complétez vos informations
    Saisissez votre nom et votre numéro (optionnels) et choisissez si vous voulez rester anonyme auprès du bénéficiaire.
  4. Validez le paiement
    Un récapitulatif affiche le montant total, les frais de service (3,5 %) et l’option de prendre les frais à votre charge. Vous confirmez le paiement via les moyens disponibles.
  5. Réception du code
    Chaque bénéficiaire reçoit par SMS un code numérique unique, valable pour un usage chez les marchands partenaires, pendant une durée définie (par exemple 90 jours).

3. Utiliser un bon chez un marchand partenaire

Les bons Nimwema sont utilisables uniquement chez les marchands partenaires affichés sur le site.

  1. Présentez votre code
    Au moment du passage en caisse, donnez le code reçu par SMS au caissier.
  2. Validation du bon
    Le caissier saisit le code dans l’interface marchande Nimwema, qui vérifie si le bon est valide, non expiré et non utilisé.
  3. Utilisation unique
    Une fois validé, le bon est marqué comme utilisé. Il ne peut plus être réutilisé ou re-généré avec le même code.
  4. Transparence
    Le marchand et Nimwema disposent d’un historique des bons utilisés pour faciliter le règlement et le suivi des flux.

4. Pour le Gouvernement, les entreprises, ONG et églises

Nimwema simplifie la gestion des aides et avantages en nature.

  • Distribution de bons à des équipes, bénéficiaires ou membres de communauté.
  • Suivi des bons émis, utilisés ou expirés via un tableau de bord dédié.
  • Rapports exportables pour la comptabilité et la réconciliation.

Pour en savoir plus sur les solutions entreprises, contactez-nous via la page Contact.
